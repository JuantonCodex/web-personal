Miralavida

Web Site

Sitio web para el lanzamiento de el nuevo Kia Cerato 2014.

Se conecta a la API de facebook para registrar la usuario y pueda participar de la interactividad de los videos del sitio.

Cada vez que el usuario contesta de manera afirmativa laspreguntas en cada video, una mensahe y una imagen del video respectivo es publicado en su muro de facebook.

Compatibilidad:

IE8+, Chrome y Firefox.

Tecnologìas:

HTML5, MySQL, Facebook API.
